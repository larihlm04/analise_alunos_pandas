# analise_alunos_pandas
Projeto com python e pandas

Análise de Desempenho de Alunos com Python e Pandas

Objetivo do projeto: Este projeto realiza uma análise simples de uma turma de alunos utilizando Python e Pandas, calculando métricas como média, aprovação e desempenho geral.

Dados usados: O dataset contém nomes de alunos e suas respectivas notas.

Tarefas feitas: cálculo da média da turma
identificação de alunos aprovados (nota ≥ 6)
identificação de reprovados (nota < 6)
maior e menor nota
contagem total de alunos

Tecnologias usadas: Python e Pandas.

O que eu aprendi:Prática com DataFrames, filtros condicionais e funções de agregação como mean, max e min.

Como executar:Instale a biblioteca Pandas e execute o arquivo Python principal.
